# BookBazaar 

## Overview

BookBazaar AI is a web-based platform designed to help students buy and sell used books easily. The platform organizes books into different categories and provides AI-based features for recommendations and summaries.

## Features

* Home Page
* Academic Books Section
* Competitive Exam Books Section
* Novel Books Section
* School Books Section
* Sell a Book Page
* AI Recommendation Chatbot (Prototype)
* AI Summary Feature (Prototype)

## Technology Stack

## How to Run

1. Download the project files.
2. Open the project folder in VS Code.
3. Install the Live Server extension.
4. Right-click on Home Page.html.
5. Click "Open with Live Server".
